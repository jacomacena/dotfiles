# Dotfiles

Steps:

\# chmod +x install.sh

\# ./install.sh -i

## Window managers

- i3wm: `.config/i3/config`
- Hyprland: `.config/hypr/hyprland.conf`
- SDDM/session selection: configured by `pos_arch.sh` through `~/.dmrc`
